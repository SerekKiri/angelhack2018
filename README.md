# Docugram

### Project description in [story.md](https://github.com/SerekKiri/angelhack2018/blob/master/STORY.md)

To run this app you need:

Go to folder:
```
cd frontend/docugram
```
Install dependencies
```
 npm install
```
And now:
```
npm run dev
```
Enjoy :)
